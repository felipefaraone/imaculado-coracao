# UI Kit — Site da Paróquia Imaculado Coração de Maria

Recriação do site institucional em React + JSX. Componentes modulares, focados em orientação pastoral e estética de manuscrito impresso.

## Componentes
- `SiteHeader.jsx` — cabeçalho sticky com marca tipográfica + navegação
- `SiteFooter.jsx` — rodapé com horários, endereço, créditos claretianos
- `HeroOpening.jsx` — abertura editorial com letra capitular e filete ornamental
- `MassSchedule.jsx` — rubrica de horários de missa (pergaminho)
- `SacramentCard.jsx` — cartão para iniciar sacramento (batismo, casamento, confissão, etc)
- `DoctrineQuote.jsx` — citação com âncora doutrinária obrigatória
- `IntentionJourney.jsx` — grade de "por que você veio?" (convertê-se, assistir, casar…)
- `HistoryChapter.jsx` — página de história em duas colunas com numeral romano
- `Divider.jsx` — separador ornamental (simples, fleuron, tríplice)

## Telas demonstradas em `index.html`
1. Home — hero + intenções + próximas missas + chamada à confissão
2. Página "A Paróquia" (história) — letra capitular, chapter num, citações
3. Página "Sacramentos" — grid de cartões
